# Core NATS (базовый)

Core NATS — это базовая функциональность системы NATS. Он работает по модели publish‑subscribe с адресацией на основе subjects/topics. Эта модель дает два ключевых преимущества: независимость от расположения и коммуникационный паттерн many‑to‑many (M:N) по умолчанию. Эти фундаментальные концепции позволяют реализовывать мощные и инновационные решения для типовых задач разработки, таких как микросервисы, без дополнительных технологий вроде балансировщиков нагрузки, API‑шлюзов или конфигурации DNS.

Системы NATS можно расширить с помощью [JetStream](../../nats-concepts/jetstream), который добавляет персистентность. В то время как Core NATS обеспечивает best‑effort доставку сообщений «не более одного раза», JetStream вводит семантику «как минимум один раз» и «ровно один раз».
