# Администрирование и использование через CLI

После запуска сервера пора пользоваться инструментом управления.  
См. [раздел установки в readme](https://github.com/nats-io/natscli?tab=readme-ov-file#installation).

```
nats --help
nats cheat
```

Мы пройдем вышеописанный сценарий и познакомим с возможностями CLI и JetStream, воспроизводя настройки.

На протяжении примера будем показывать другие команды, такие как `nats pub` и `nats sub`, чтобы взаимодействовать с системой. Это обычные команды core NATS, и JetStream полностью пригоден к использованию только через core NATS.

Мы коснемся некоторых дополнительных возможностей, но, пожалуйста, ознакомьтесь с разделом о модели дизайна, чтобы понять все возможные варианты.
