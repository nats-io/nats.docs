## NATS Tools

В экосистеме NATS есть много инструментов для настройки сервера, улучшения мониторинга и оптимизации производительности:

- [nats](nats_tools/natscli.md) - Взаимодействие с NATS и управление им
- [nk](nats_tools/nk.md) - Генерация NKeys
- [nsc](nats_tools/nsc/README.md) - Настройка операторов, аккаунтов и пользователей
- [nats account server](nats_tools/nas/README.md) - Выдача JWT аккаунтов
- [nats top](nats_tools/nats_top/README.md) - Мониторинг NATS Server
- [nats-bench](nats_tools/natsbench.md) - Бенчмарк NATS Server
- [prometheus-nats-exporter](https://github.com/nats-io/prometheus-nats-exporter) - Экспорт метрик NATS Server в [Prometheus](https://prometheus.io/) и на дашборд [Grafana](https://grafana.com).
